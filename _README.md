# 🇳🇬 Detecting AI-Generated Political Misinformation in Nigeria
## AISIP Capstone — Pathway 4: AI Engineering (Option B: ML Research Notebook)

---

## Research Question
> Can machine learning models reliably distinguish AI-generated political misinformation
> from genuine Nigerian political discourse?

---

## Problem Statement
Nigeria is Africa's most populous democracy with 200M+ citizens and one of the
continent's most active social media ecosystems. During the 2023 general elections,
researchers documented widespread AI-generated synthetic media, fabricated quotes
attributed to political figures, and coordinated inauthentic behaviour across WhatsApp,
Twitter/X, and Facebook. This project builds an ML pipeline to detect such content.

## Key Features Engineered

**Stylometric (14 features):**
- Character count, word count, sentence count
- Average word length, average sentence length
- Lexical diversity (unique words / total words)
- Stopword ratio, punctuation ratio
- Number of numerical claims
- Exclamation marks, question marks, CAPS words

**TF-IDF (300 features):**
- Unigrams and bigrams
- Sublinear TF scaling
- English stopwords removed
- Minimum document frequency = 2

---

## Key Findings
- AI-generated texts contain significantly more numerical claims and higher lexical diversity
- Human texts use more exclamation marks and informal/emotional language
- Combined stylometric + TF-IDF features outperform either set alone (ablation confirmed)
- Neural network competitive but ensemble methods perform similarly on this dataset size

---

## Limitations
1. Dataset size: 50 samples per class — production needs 10,000+ verified examples
2. Language gap: Nigerian Pidgin and code-switching underrepresented
3. Adversarial: LLMs improve constantly — requires continuous retraining
4. Ethical: False positives could suppress legitimate political speech

## Model Card

**Intended use:** Research tool for fact-checkers and election monitors in Nigeria

**Who should use:** Civil society, researchers, journalists as a triage/prioritisation tool

**Who should NOT use:** Governments for automated censorship; platforms for auto-removal without human review

**Input:** Any Nigerian political text string

**Output:** Binary classification — Human (0) or AI-Generated (1) with probability score
