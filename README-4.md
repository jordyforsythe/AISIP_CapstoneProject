## 🎯 What This Project Does

**NaijaFact** detects whether Nigerian political text was written by a **human**
or generated by an **AI** (e.g. ChatGPT, GPT-4). It addresses the growing threat of
AI-generated political misinformation in Nigeria's digital information ecosystem —
especially during election periods when synthetic content has historically contributed
to electoral mistrust and post-election violence.

## 🧠 ML Pipeline

```
Raw Text Input (Nigerian English / Pidgin / mixed)
        ↓
Feature Extraction
  ├── 14 Stylometric features (word count, sentence length,
  │   lexical diversity, punctuation, CAPS, exclamations…)
  ├── Nigerian Pidgin detector (25 expressions)
  └── Authority/Fabrication marker detector (40 phrases)
        ↓
Weighted Scoring Classifier
  ├── Human signals: Pidgin (-0.12/hit), Exclamations (-0.07)
  └── AI signals: Authority markers (+0.16/hit), Numbers (+0.09/claim)
        ↓
Binary Output: Human (0) | AI-Generated (1) + Confidence %
        ↓
Signal Breakdown + Recommended Actions
        ↓
Auto-saved to CSV → Downloadable for retraining
```
